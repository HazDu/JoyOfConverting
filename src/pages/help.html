How to use

Overview

This converter is used to convert .paint files into .png files or .png files into .paint files. The .paint file format is a file format that's used by the Joy of Painting Minecraft mod. The mod allows you to import and export .paint files and to enjoy your paintings even outside of minecraft or import pixel arts, you can change the format between .paint and .png.

How to use the converter

It's pretty easy! Just drag and drop a .png or .paint file into the drop area on the Home-Page. The converter will then automatically forward you to one of two sites.

The .paint to .png site:
On this site you'll see an input box. In there you write the name of the converted image file. E.g. "Cool Painting" will give you a file with the name "Cool Painting.png". After putting in the text, click the "Convert" button and your image will be converted and automatically downloaded (This may take up to a few seconds).

The .png to .paint site:
On this site you'll see two input boxes the one labeled with "Title" will set the signed title of your ingame item, as well as the file name of your converted file and the one labeled "Author" will set the Author's name of the ingame item. After you made the input you'll need to click "Convert" and your image will be converted and automatically downloaded (This may take up to a few seconds).

Import / Export .paint files in minecraft

To IMPORT a .paint file you'll first need to put it in the "paintings" folder, the mod created in your minecraft directory. Then you're going to open the chat ingame and type [/paintimport FileName] and type in just the file name without the ".paint" file type. If you are in survival mode you'll need to have an empty canvas of the right size in your main hand and a full palette in your offhand.

To EXPORT a .paint file you'll need to open the chat ingame and type [/paintexport FileName] and type in just the file name without the ".paint" file type. After exporting, you'll find the .paint file in the "paintings" folder, the mod created in your minecraft directory.
